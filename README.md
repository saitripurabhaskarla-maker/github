# Simple Interest Calculator

## Project Name
**Simple Interest Calculator** — a Bash-based command-line tool for computing simple interest.

## Description
This repository contains a simple, beginner-friendly Bash script that calculates simple interest based on three user-supplied inputs:

- **Principal amount** — the initial sum of money
- **Rate of interest** — the annual interest rate (in percent)
- **Time period** — the duration in years

The project was built as part of a hands-on final project to demonstrate proficiency with Git and GitHub, including repository structure, licensing, community health files, forking, branching, and merging workflows.

## Formula
Simple Interest is calculated using the standard formula:

```
SI = (P × R × T) / 100
```

Where:
- `P` = Principal
- `R` = Rate of Interest
- `T` = Time Period (in years)

## Usage
```bash
chmod +x simple-interest.sh
./simple-interest.sh
```

You will be prompted to enter the principal, rate of interest, and time period. The script will then output the calculated simple interest.
